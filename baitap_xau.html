Bài Tập Xử Lý Xâu

Bộ đề rèn luyện kỹ năng Xử lý Xâu & Xâu Số dành cho HSG Tin học

Bài 1: ĐẾM TỪ
DEMTU.INP / DEMTU.OUT

Cho xâu S gồm các chữ cái (hoa và thường) và dấu cách. Hãy đếm số từ trong xâu. Từ là nhóm ký tự liên tiếp không phải dấu cách.

Dữ liệu vào (DEMTU.INP):
  • Một dòng chứa xâu S (1 ≤ |S| ≤ 106)
Kết quả ra (DEMTU.OUT):
  • Một số nguyên duy nhất — số từ trong xâu

DEMTU.INP

  Hello   World

DEMTU.OUT

2
📝 Code C++ — F1. Đếm từ 
// Bài 1: DEMTU — Đếm từ trong xâu
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DEMTU.INP");
    ofstream out("DEMTU.OUT");
    string s;
    getline(in, s);
    int cnt = 0;
    bool inWord = false;
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] != ' ') {
            if (!inWord) { cnt++; inWord = true; }
        } else {
            inWord = false;
        }
    }
    out << cnt;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Duyệt xâu, dùng biến inWord theo dõi trạng thái. Khi gặp ký tự không phải dấu cách mà chưa trong từ → tăng đếm. Độ phức tạp: O(|S|).
Bài 2: CHUẨN HÓA XÂU
CHUANHOA.INP / CHUANHOA.OUT

Cho xâu S gồm chữ cái (hoa và thường) và dấu cách (có thể thừa ở đầu, cuối, giữa các từ). Hãy chuẩn hóa xâu: bỏ dấu cách thừa (đầu, cuối và giữa các từ chỉ giữ 1 dấu cách), viết hoa chữ cái đầu mỗi từ, các chữ còn lại viết thường.

Dữ liệu vào (CHUANHOA.INP):
  • Một dòng chứa xâu S (1 ≤ |S| ≤ 106)
Kết quả ra (CHUANHOA.OUT):
  • Một dòng — xâu đã chuẩn hóa

CHUANHOA.INP

  hello   world

CHUANHOA.OUT

Hello World
📝 Code C++ — F2. Chuẩn hóa xâu 
// Bài 2: CHUANHOA — Chuẩn hóa xâu
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CHUANHOA.INP");
    ofstream out("CHUANHOA.OUT");
    string s;
    getline(in, s);
    vector<string> words;
    string w;
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] != ' ') {
            w += s[i];
        } else {
            if (!w.empty()) { words.push_back(w); w = ""; }
        }
    }
    if (!w.empty()) words.push_back(w);
    for (int i = 0; i < (int)words.size(); i++) {
        for (int j = 0; j < (int)words[i].size(); j++)
            words[i][j] = tolower(words[i][j]);
        words[i][0] = toupper(words[i][0]);
        if (i > 0) out << " ";
        out << words[i];
    }
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Tách từ, chuyển tất cả thành chữ thường, viết hoa chữ cái đầu mỗi từ, nối lại bằng 1 dấu cách. Độ phức tạp: O(|S|).

DAOTU.INP / DAOTU.OUT

Cho xâu S gồm các từ cách nhau bởi dấu cách. In ra xâu mới với thứ tự các từ bị đảo ngược (từ cuối lên đầu).

Dữ liệu vào (DAOTU.INP):
  • Một dòng chứa xâu S (1 ≤ |S| ≤ 106)
Kết quả ra (DAOTU.OUT):
  • Một dòng — xâu với thứ tự từ đảo ngược

DAOTU.INP

hello world test

DAOTU.OUT

test world hello
📝 Code C++ — F3. Đảo từ 
// Bài 3: DAOTU — Đảo ngược thứ tự từ
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DAOTU.INP");
    ofstream out("DAOTU.OUT");
    string s;
    getline(in, s);
    vector<string> words;
    string w;
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] != ' ') {
            w += s[i];
        } else {
            if (!w.empty()) { words.push_back(w); w = ""; }
        }
    }
    if (!w.empty()) words.push_back(w);
    for (int i = (int)words.size() - 1; i >= 0; i--) {
        if (i < (int)words.size() - 1) out << " ";
        out << words[i];
    }
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Tách từ vào mảng, in ngược từ cuối lên đầu, cách nhau bởi 1 dấu cách. Độ phức tạp: O(|S|).
Bài 4: ĐỐI XỨNG
DOIXUNG.INP / DOIXUNG.OUT

Cho T xâu, mỗi xâu chỉ gồm chữ cái thường. Với mỗi xâu, kiểm tra xâu đó có phải xâu đối xứng (palindrome) không. In "YES" hoặc "NO".

Dữ liệu vào (DOIXUNG.INP):
  • Dòng 1: Số nguyên T (1 ≤ T ≤ 100)
  • T dòng tiếp theo: Mỗi dòng một xâu S (1 ≤ |S| ≤ 105)
Kết quả ra (DOIXUNG.OUT):
  • T dòng, mỗi dòng "YES" hoặc "NO"

3
abcba
hello
racecar

DOIXUNG.OUT

NO
YES
📝 Code C++ — F4. Palindrome 
// Bài 4: DOIXUNG — Kiểm tra xâu đối xứng
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DOIXUNG.INP");
    ofstream out("DOIXUNG.OUT");
    int t;
    in >> t;
    while (t--) {
        string s;
        in >> s;
        bool ok = true;
        for (int i = 0; i < (int)s.size() / 2; i++)
            if (s[i] != s[s.size() - 1 - i]) { ok = false; break; }
        out << (ok ? "YES" : "NO") << "\n";
    }
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: So sánh ký tự đầu–cuối tiến vào giữa. Nếu tất cả bằng nhau → đối xứng. Độ phức tạp: O(|S|) mỗi xâu.
Bài 5: MÃ HÓA CAESAR
CAESAR.INP / CAESAR.OUT

Cho số nguyên K và xâu S. Mã hóa xâu bằng phép dịch Caesar: mỗi chữ cái được dịch K vị trí trong bảng chữ cái (xoay vòng). Các ký tự không phải chữ cái giữ nguyên.

Dữ liệu vào (CAESAR.INP):
  • Dòng 1: Số nguyên K (0 ≤ K ≤ 25)
  • Dòng 2: Xâu S (1 ≤ |S| ≤ 106)
Kết quả ra (CAESAR.OUT):
  • Một dòng — xâu đã mã hóa

CAESAR.INP

3
Hello World!

CAESAR.OUT

Khoor Zruog!
📝 Code C++ — F5. Caesar Cipher 
// Bài 5: CAESAR — Mã hóa Caesar
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CAESAR.INP");
    ofstream out("CAESAR.OUT");
    int k;
    in >> k;
    in.ignore();
    string s;
    getline(in, s);
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] >= 'A' && s[i] <= 'Z')
            s[i] = 'A' + (s[i] - 'A' + k) % 26;
        else if (s[i] >= 'a' && s[i] <= 'z')
            s[i] = 'a' + (s[i] - 'a' + k) % 26;
    }
    out << s;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dịch mỗi chữ cái K vị trí trong bảng chữ cái (mod 26), giữ nguyên hoa/thường. Ký tự không phải chữ cái giữ nguyên. Độ phức tạp: O(|S|).
Bài 6: NÉN XÂU
NENXAU.INP / NENXAU.OUT

Cho xâu S gồm chữ cái thường. Nén xâu theo phương pháp RLE (Run-Length Encoding): thay mỗi nhóm ký tự liên tiếp giống nhau bằng ký tự đó kèm số lần lặp.

Dữ liệu vào (NENXAU.INP):
  • Một dòng chứa xâu S (1 ≤ |S| ≤ 106, chỉ chữ cái thường)
Kết quả ra (NENXAU.OUT):
  • Một dòng — xâu đã nén (ví dụ: "aaabb" → "a3b2")

NENXAU.INP

aaabbcdddd

NENXAU.OUT

a3b2c1d4
📝 Code C++ — F6. Run-Length Encoding 
// Bài 6: NENXAU — Nén xâu RLE
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("NENXAU.INP");
    ofstream out("NENXAU.OUT");
    string s;
    in >> s;
    int i = 0;
    while (i < (int)s.size()) {
        char c = s[i];
        int cnt = 0;
        while (i < (int)s.size() && s[i] == c) { cnt++; i++; }
        out << c << cnt;
    }
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Duyệt xâu, đếm số lần liên tiếp của mỗi ký tự, ghi ra ký tự kèm số lần. Độ phức tạp: O(|S|).
Bài 7: KÝ TỰ XUẤT HIỆN NHIỀU NHẤT
KITUMAX.INP / KITUMAX.OUT

Cho xâu S gồm chữ cái thường. Tìm ký tự xuất hiện nhiều nhất và số lần xuất hiện của nó. Nếu có nhiều ký tự cùng tần suất, lấy ký tự xuất hiện đầu tiên trong xâu.

Dữ liệu vào (KITUMAX.INP):
  • Một dòng chứa xâu S (1 ≤ |S| ≤ 106, chỉ chữ cái thường)
Kết quả ra (KITUMAX.OUT):
  • Một dòng gồm ký tự và số lần xuất hiện, cách nhau bởi dấu cách

KITUMAX.INP

abracadabra

KITUMAX.OUT

a 5
📝 Code C++ — F7. Tần suất ký tự 
// Bài 7: KITUMAX — Ký tự xuất hiện nhiều nhất
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("KITUMAX.INP");
    ofstream out("KITUMAX.OUT");
    string s;
    in >> s;
    int cnt[26] = {};
    for (int i = 0; i < (int)s.size(); i++) cnt[s[i] - 'a']++;
    char best = s[0];
    for (int i = 0; i < (int)s.size(); i++)
        if (cnt[s[i] - 'a'] > cnt[best - 'a']) best = s[i];
    out << best << " " << cnt[best - 'a'];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Đếm tần suất 26 chữ cái. Duyệt lại xâu tìm ký tự đầu tiên có tần suất lớn nhất. Độ phức tạp: O(|S|).

Kết quả ra (CONGSO.OUT):
  • Một dòng — kết quả A + B

CONGSO.INP

999
1

CONGSO.OUT

1000
📝 Code C++ — F8. Cộng xâu số 
// Bài 8: CONGSO — Cộng hai số lớn dạng xâu
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CONGSO.INP");
    ofstream out("CONGSO.OUT");
    string a, b;
    in >> a >> b;
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    string result;
    int carry = 0;
    for (int i = 0; i < (int)a.size() || i < (int)b.size(); i++) {
        int x = (i < (int)a.size()) ? a[i] - '0' : 0;
        int y = (i < (int)b.size()) ? b[i] - '0' : 0;
        int sum = x + y + carry;
        result += (char)('0' + sum % 10);
        carry = sum / 10;
    }
    if (carry) result += '1';
    reverse(result.begin(), result.end());
    out << result;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Đảo ngược hai xâu, cộng từng chữ số từ hàng đơn vị kèm nhớ, rồi đảo kết quả. Độ phức tạp: O(max(|A|, |B|)).

📝 Code C++ — F9. Trừ xâu số 
// Bài 9: TRUSO — Trừ hai số lớn (A ≥ B)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("TRUSO.INP");
    ofstream out("TRUSO.OUT");
    string a, b;
    in >> a >> b;
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    string result;
    int borrow = 0;
    for (int i = 0; i < (int)a.size(); i++) {
        int x = a[i] - '0';
        int y = (i < (int)b.size()) ? b[i] - '0' : 0;
        int diff = x - y - borrow;
        if (diff < 0) { diff += 10; borrow = 1; }
        else borrow = 0;
        result += (char)('0' + diff);
    }
    while (result.size() > 1 && result.back() == '0') result.pop_back();
    reverse(result.begin(), result.end());
    out << result;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Đảo ngược hai xâu, trừ từng chữ số kèm mượn. Cuối cùng xóa số 0 thừa ở đầu kết quả. Độ phức tạp: O(|A|).

📝 Code C++ — F10. Nhân xâu số 
// Bài 10: NHANSO — Nhân số lớn với số bé
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("NHANSO.INP");
    ofstream out("NHANSO.OUT");
    string a;
    int b;
    in >> a >> b;
    if (b == 0) { out << "0"; return 0; }
    reverse(a.begin(), a.end());
    string result;
    long long carry = 0;
    for (int i = 0; i < (int)a.size(); i++) {
        long long prod = (long long)(a[i] - '0') * b + carry;
        result += (char)('0' + prod % 10);
        carry = prod / 10;
    }
    while (carry > 0) {
        result += (char)('0' + carry % 10);
        carry /= 10;
    }
    reverse(result.begin(), result.end());
    out << result;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Đảo xâu A, nhân từng chữ số với B kèm nhớ (dùng long long tránh tràn), rồi đảo kết quả. Độ phức tạp: O(|A|).

  • Dòng 1: Thương A div B (không có số 0 thừa ở đầu)
  • Dòng 2: Dư A mod B

CHIASO.OUT

📝 Code C++ — F11. Chia xâu số 
// Bài 11: CHIASO — Chia số lớn cho số bé
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CHIASO.INP");
    ofstream out("CHIASO.OUT");
    string a;
    int b;
    in >> a >> b;
    string quotient;
    long long remainder = 0;
    for (int i = 0; i < (int)a.size(); i++) {
        remainder = remainder * 10 + (a[i] - '0');
        quotient += (char)('0' + (int)(remainder / b));
        remainder %= b;
    }
    // Xóa số 0 thừa ở đầu
    int start = 0;
    while (start < (int)quotient.size() - 1 && quotient[start] == '0') start++;
    quotient = quotient.substr(start);
    out << quotient << "\n" << remainder;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Chia dần từ trái sang phải: dư = dư × 10 + chữ_số; thương += dư/B; dư = dư%B. Dùng long long cho dư (vì dư × 10 có thể vượt int). Độ phức tạp: O(|A|).
Bài 12: SO SÁNH SỐ LỚN
SOSANHSO.INP / SOSANHSO.OUT

Cho T cặp số nguyên không âm rất lớn (dạng xâu chữ số). Với mỗi cặp (A, B), so sánh A và B, in ra "<", ">" hoặc "=".

Dữ liệu vào (SOSANHSO.INP):
  • Dòng 1: Số nguyên T (1 ≤ T ≤ 100)
  • T dòng tiếp theo: Mỗi dòng hai xâu A, B cách nhau dấu cách (1 ≤ |A|, |B| ≤ 106)
Kết quả ra (SOSANHSO.OUT):
  • T dòng, mỗi dòng "<", ">" hoặc "="

999 1000

📝 Code C++ — F12. So sánh xâu số 
// Bài 12: SOSANHSO — So sánh hai số lớn
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("SOSANHSO.INP");
    ofstream out("SOSANHSO.OUT");
    int t;
    in >> t;
    while (t--) {
        string a, b;
        in >> a >> b;
        // Xóa số 0 thừa ở đầu
        while (a.size() > 1 && a[0] == '0') a = a.substr(1);
        while (b.size() > 1 && b[0] == '0') b = b.substr(1);
        if (a.size() != b.size())
            out << (a.size() < b.size() ? "<" : ">") << "\n";
        else {
            if (a < b) out << "<\n";
            else if (a > b) out << ">\n";
            else out << "=\n";
        }
    }
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Xóa số 0 thừa ở đầu. So sánh theo độ dài trước: dài hơn thì lớn hơn. Nếu bằng nhau thì so chuỗi từ trái sang phải (so sánh từ điển trùng với so sánh số khi cùng độ dài). Độ phức tạp: O(|A| + |B|).

#include <bits/stdc++.h>
using namespace std;

string multiply(string a, int b) {
    reverse(a.begin(), a.end());
    string result;
    long long carry = 0;
    for (int i = 0; i < (int)a.size(); i++) {
        long long prod = (long long)(a[i] - '0') * b + carry;
        result += (char)('0' + prod % 10);
        carry = prod / 10;
    }
    while (carry > 0) {
        result += (char)('0' + carry % 10);
        carry /= 10;
    }
    reverse(result.begin(), result.end());
    return result;
}

int main() {
    ifstream in("GTSO.INP");
    ofstream out("GTSO.OUT");
    int n;
    in >> n;
    string result = "1";
    for (int i = 2; i <= n; i++)
        result = multiply(result, i);
    out << result;
    in.close();
    out.close();
    return 0;
}

// Bài 14: FIBONACCI — Fibonacci số lớn (dùng xâu)
#include <bits/stdc++.h>
using namespace std;

string add(string a, string b) {
    reverse(a.begin(), a.end());
    reverse(b.begin(), b.end());
    string result;
    int carry = 0;
    for (int i = 0; i < (int)a.size() || i < (int)b.size(); i++) {
        int x = (i < (int)a.size()) ? a[i] - '0' : 0;
        int y = (i < (int)b.size()) ? b[i] - '0' : 0;
        int sum = x + y + carry;
        result += (char)('0' + sum % 10);
        carry = sum / 10;
    }
    if (carry) result += '1';
    reverse(result.begin(), result.end());
    return result;
}

int main() {
    ifstream in("FIBONACCI.INP");
    ofstream out("FIBONACCI.OUT");
    int n;
    in >> n;
    if (n <= 1) { out << n; return 0; }
    string a = "0", b = "1";
    for (int i = 2; i <= n; i++) {
        string c = add(a, b);
        a = b;
        b = c;
    }
    out << b;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Cộng dồn hai xâu số lớn: F(i) = F(i−1) + F(i−2). Hàm add cộng hai xâu chữ số. Độ phức tạp: O(N × số_chữ_số).

// Bài 16: DEMXAUCON — Đếm số lần xuất hiện xâu con (cho phép chồng)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DEMXAUCON.INP");
    ofstream out("DEMXAUCON.OUT");
    string s, p;
    in >> s >> p;
    int cnt = 0, n = s.size(), m = p.size();
    for (int i = 0; i <= n - m; i++) {
        bool ok = true;
        for (int j = 0; j < m; j++)
            if (s[i + j] != p[j]) { ok = false; break; }
        if (ok) cnt++;
    }
    out << cnt;
    in.close();
    out.close();
    return 0;
}

// Bài 17: ANAGRAM — Kiểm tra hoán vị ký tự
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("ANAGRAM.INP");
    ofstream out("ANAGRAM.OUT");
    int t;
    in >> t;
    while (t--) {
        string a, b;
        in >> a >> b;
        if (a.size() != b.size()) { out << "NO\n"; continue; }
        int cnt[26] = {};
        for (int i = 0; i < (int)a.size(); i++) {
            cnt[a[i] - 'a']++;
            cnt[b[i] - 'a']--;
        }
        bool ok = true;
        for (int i = 0; i < 26; i++)
            if (cnt[i] != 0) { ok = false; break; }
        out << (ok ? "YES" : "NO") << "\n";
    }
    in.close();
    out.close();
    return 0;
}

// Bài 18: NGOACHOPLE — Kiểm tra ngoặc hợp lệ bằng stack
#include <bits/stdc++.h>
using namespace std;

char matchOf(char c) {
    if (c == ')') return '(';
    if (c == ']') return '[';
    return '{';
}

int main() {
    ifstream in("NGOACHOPLE.INP");
    ofstream out("NGOACHOPLE.OUT");
    int t;
    in >> t;
    while (t--) {
        string s;
        in >> s;
        stack<char> st;
        bool ok = true;
        for (int i = 0; i < (int)s.size() && ok; i++) {
            if (s[i] == '(' || s[i] == '[' || s[i] == '{') {
                st.push(s[i]);
            } else {
                if (st.empty() || st.top() != matchOf(s[i])) ok = false;
                else st.pop();
            }
        }
        if (!st.empty()) ok = false;
        out << (ok ? "YES" : "NO") << "\n";
    }
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng stack: gặp ngoặc mở → đẩy vào; gặp ngoặc đóng → kiểm tra đỉnh stack có khớp loại không. Cuối cùng stack phải rỗng. Độ phức tạp: O(|S|).

// Bài 19: SAPXEPTU — Sắp xếp từ theo thứ tự từ điển
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("SAPXEPTU.INP");
    ofstream out("SAPXEPTU.OUT");
    string s;
    getline(in, s);
    vector<string> words;
    string w;
    for (int i = 0; i < (int)s.size(); i++) {
        if (s[i] != ' ') {
            w += s[i];
        } else {
            if (!w.empty()) { words.push_back(w); w = ""; }
        }
    }
    if (!w.empty()) words.push_back(w);
    stable_sort(words.begin(), words.end());
    for (int i = 0; i < (int)words.size(); i++) {
        if (i > 0) out << " ";
        out << words[i];
    }
    in.close();
    out.close();
    return 0;
}

// Bài 20: CHUYENCS — Chuyển hệ cơ số (số lớn dạng xâu)
#include <bits/stdc++.h>
using namespace std;

// Chia xâu số (hệ 10) cho b, trả về thương (xâu) và dư
pair<string, int> divmod(string a, int b) {
    string quotient;
    int remainder = 0;
    for (int i = 0; i < (int)a.size(); i++) {
        remainder = remainder * 10 + (a[i] - '0');
        quotient += (char)('0' + remainder / b);
        remainder %= b;
    }
    int start = 0;
    while (start < (int)quotient.size() - 1 && quotient[start] == '0') start++;
    quotient = quotient.substr(start);
    return make_pair(quotient, remainder);
}

int main() {
    ifstream in("CHUYENCS.INP");
    ofstream out("CHUYENCS.OUT");
    string n;
    int b;
    in >> n >> b;
    if (n == "0") { out << "0"; return 0; }
    string digits = "0123456789ABCDEF";
    string result;
    while (n != "0") {
        pair<string, int> dm = divmod(n, b);
        result += digits[dm.second];
        n = dm.first;
    }
    reverse(result.begin(), result.end());
    out << result;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Chia liên tục N cho B, lấy dư (chữ số hệ B), cho đến khi N = 0. Hàm divmod chia xâu số hệ 10 cho B (xử lý số lớn). Đảo kết quả để có thứ tự đúng. Độ phức tạp: O(|N|² / log B).
