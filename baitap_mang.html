Bài 1: ĐIỆN NĂNG
DIENNANG.INP / DIENNANG.OUT

Công ty điện lực thành phố quản lý N trạm biến áp được đánh số từ 1 đến N. Mỗi trạm thứ i tiêu thụ Ai đơn vị điện năng mỗi ngày.

Hàng ngày, giám đốc cần báo cáo tổng điện năng tiêu thụ của các trạm trong một khu vực liên tiếp. Có Q báo cáo cần thực hiện, báo cáo thứ j yêu cầu tính tổng điện năng của các trạm từ vị trí Lj đến Rj.

Dữ liệu vào (DIENNANG.INP):
  • Dòng 1: Hai số nguyên N và Q (1 ≤ N, Q ≤ 105)
  • Dòng 2: N số nguyên A1, A2, ..., AN (|Ai| ≤ 109)
  • Q dòng tiếp theo: Mỗi dòng chứa hai số Lj và Rj (1 ≤ Lj ≤ Rj ≤ N)
Kết quả ra (DIENNANG.OUT):
  • Gồm Q dòng, dòng thứ j là tổng điện năng từ trạm Lj đến trạm Rj

// Bài 1: DIENNANG - Tiền tố
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5 + 5;
long long a[MAXN], prefix[MAXN];
int n, q;

int main() {
    ifstream in("DIENNANG.INP");
    ofstream out("DIENNANG.OUT");
    
    in >> n >> q;
    
    // Nhập mảng và tính prefix sum
    prefix[0] = 0;
    for(int i = 1; i <= n; i++) {
        in >> a[i];
        prefix[i] = prefix[i-1] + a[i];
    }
    
    // Xử lý Q truy vấn
    while(q--) {
        int l, r;
        in >> l >> r;
        // Tổng [l, r] = prefix[r] - prefix[l-1]
        out << prefix[r] - prefix[l-1] << "\n";
    }
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Prefix Sum (A1). Hình dung: prefix[i] như "tổng tiền từ đầu dãy đến vị trí i". Muốn tính tổng đoạn [l, r]? Lấy tổng đến r trừ tổng đến l−1: prefix[r] − prefix[l−1]. Giống như muốn biết số tiền từ ngày 3 đến ngày 7, lấy "tổng 7 ngày" trừ "tổng 2 ngày đầu". Tính prefix 1 lần O(N), sau đó mỗi truy vấn chỉ O(1).
Bài 2: CHIA BÁN
CHIABAN.INP / CHIABAN.OUT

Một cửa hàng có N sản phẩm xếp thành hàng, sản phẩm thứ i có giá Ai đồng. Chủ cửa hàng muốn chọn một đoạn sản phẩm liên tiếp để bán cho khách hàng sỉ. Khách hàng sỉ yêu cầu tổng giá trị đơn hàng phải chia hết cho K.

Hỏi có bao nhiêu cách chọn đoạn sản phẩm liên tiếp thỏa mãn yêu cầu?

Dữ liệu vào (CHIABAN.INP):
  • Dòng 1: Hai số nguyên N và K (1 ≤ N ≤ 105; 1 ≤ K ≤ 106)
  • Dòng 2: N số nguyên dương A1, A2, ..., AN (Ai ≤ 109)
Kết quả ra (CHIABAN.OUT):
  • Một số nguyên duy nhất là số cách chọn đoạn sản phẩm liên tiếp có tổng chia hết cho K

// Bài 2: CHIABAN - Prefix Sum + Đếm dư
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CHIABAN.INP");
    ofstream out("CHIABAN.OUT");
    
    int n, k;
    in >> n >> k;
    
    // cnt[r] = số prefix sum có dư r khi chia cho k
    vector<long long> cnt(k, 0);
    cnt[0] = 1; // prefix[0] = 0 có dư 0
    
    long long prefix = 0, ans = 0;
    for(int i = 0; i < n; i++) {
        long long a;
        in >> a;
        prefix = (prefix + a % k) % k;
        // Đoạn [j+1, i] chia hết cho k khi prefix[i] ≡ prefix[j] (mod k)
        ans += cnt[prefix];
        cnt[prefix]++;
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Kết hợp Prefix Sum (A1) + Mảng đếm (A10). Tổng đoạn [j+1, i] = prefix[i] − prefix[j]. Khi nào hiệu này chia hết K? Khi 2 prefix cùng dư khi chia K. VD: prefix[i]=11 = 3×3 + 2, prefix[j]=5 = 3×1 + 2. Trừ nhau: 11−5 = 3×(3−1) + (2−2) = 6 → phần dư triệt tiêu, hiệu luôn chia hết K! Hình dung: xếp các prefix vào "nhóm" theo số dư khi chia K: nhóm 0, nhóm 1, ..., nhóm K−1. Chọn 2 phần tử bất kì trong cùng nhóm = 1 đoạn hợp lệ. Nhóm có c phần tử → c×(c−1)/2 cặp. Cộng hết là đáp án. O(N).
Bài 3: CHẤT LƯỢNG ỔN ĐỊNH
ONDINH.INP / ONDINH.OUT

Nhà máy sản xuất N lô hàng liên tiếp, lô thứ i có điểm chất lượng Ai. Ban quản lý coi một dây chuyền sản xuất là "ổn định" nếu điểm chất lượng trung bình của các lô liên tiếp đúng bằng một ngưỡng K cho trước.

Hãy tìm dây chuyền sản xuất ổn định có số lô hàng nhiều nhất.

Dữ liệu vào (ONDINH.INP):

  • Một số nguyên duy nhất là số lô hàng nhiều nhất của dây chuyền ổn định. In 0 nếu không tồn tại.

// Bài 3: ONDINH - Prefix Sum + Map
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("ONDINH.INP");
    ofstream out("ONDINH.OUT");
    
    int n;
    long long k;
    in >> n >> k;
    
    // Biến đổi: B[i] = A[i] - K, tìm đoạn dài nhất có tổng = 0
    map<long long, int> firstPos;
    firstPos[0] = 0; // prefix[0] = 0 tại vị trí 0
    
    long long prefix = 0;
    int ans = 0;
    
    for(int i = 1; i <= n; i++) {
        long long a;
        in >> a;
        prefix += (a - k);
        
        if(firstPos.count(prefix)) {
            ans = max(ans, i - firstPos[prefix]);
        } else {
            firstPos[prefix] = i;
        }
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Biến đổi bài toán (4.1) + Prefix Sum (A1). Trung bình đoạn = K nghĩa là tổng đoạn = K × độ dài. VD: TB 5 phần tử = 3 thì tổng = 15 = 3×5. Nếu trừ mỗi phần tử đi K: B[i] = A[i] − K, thì tổng đoạn B = tổng đoạn A − K×(độ dài) = 0. Vậy bài toán thành: tìm đoạn dài nhất có tổng = 0 trên mảng B. Dùng prefix sum: nếu prefix[i] = prefix[j] thì tổng [j+1..i] = 0. Hình dung prefix như đồ thị: tìm 2 điểm cùng độ cao xa nhất. Dùng map (C1) lưu vị trí đầu tiên mỗi giá trị prefix. O(N log N).
Bài 4: RUỘNG LÚA
RUONG.INP / RUONG.OUT

Một nông trường có cánh đồng hình chữ nhật kích thước M×N ô. Mỗi ô (i,j) có năng suất Ai,j tạ/vụ. Nông trường nhận được Q đơn đặt hàng, đơn hàng thứ k yêu cầu giao toàn bộ lúa từ vùng hình chữ nhật có góc trên trái (r1, c1) và góc dưới phải (r2, c2).

Với mỗi đơn hàng, tính tổng sản lượng lúa cần giao.

Dữ liệu vào (RUONG.INP):
  • Dòng 1: Ba số nguyên M, N, Q (1 ≤ M, N ≤ 1000; 1 ≤ Q ≤ 105)
  • M dòng tiếp theo: Mỗi dòng chứa N số nguyên mô tả năng suất (|Ai,j| ≤ 106)
  • Q dòng tiếp theo: Mỗi dòng chứa 4 số r1, c1, r2, c2 (1 ≤ r1 ≤ r2 ≤ M; 1 ≤ c1 ≤ c2 ≤ N)
Kết quả ra (RUONG.OUT):
  • Gồm Q dòng, mỗi dòng là tổng sản lượng của vùng tương ứng

// Bài 4: RUONG - Prefix Sum 2D
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
long long a[MAXN][MAXN], prefix[MAXN][MAXN];

int main() {
    ifstream in("RUONG.INP");
    ofstream out("RUONG.OUT");
    
    int m, n, q;
    in >> m >> n >> q;
    
    // Nhập và tính prefix sum 2D
    for(int i = 1; i <= m; i++) {
        for(int j = 1; j <= n; j++) {
            in >> a[i][j];
            prefix[i][j] = a[i][j] + prefix[i-1][j] 
                         + prefix[i][j-1] - prefix[i-1][j-1];
        }
    }
    
    // Xử lý Q truy vấn
    while(q--) {
        int r1, c1, r2, c2;
        in >> r1 >> c1 >> r2 >> c2;
        // Công thức bao hàm loại trừ
        long long sum = prefix[r2][c2] - prefix[r1-1][c2] 
                      - prefix[r2][c1-1] + prefix[r1-1][c1-1];
        out << sum << "\n";
    }
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Prefix Sum 2D (A1). Mở rộng prefix sum lên 2 chiều: prefix[i][j] = tổng mọi ô từ góc (1,1) đến (i,j). Muốn tổng vùng HCN? Hình dung vẽ 4 hình chữ nhật lồng nhau: lấy ô lớn nhất, trừ 2 phần thừa (trên và trái), cộng lại góc bị trừ 2 lần: prefix[r2][c2] − prefix[r1−1][c2] − prefix[r2][c1−1] + prefix[r1−1][c1−1]. O(1) mỗi truy vấn, O(M×N) tiền xử lý.
Bài 5: TƯỚI CÂY
TUOICAY.INP / TUOICAY.OUT

Vườn cây có N cây xếp thành hàng, đánh số từ 1 đến N. Ban đầu tất cả cây đều chưa được tưới (lượng nước = 0). Người làm vườn thực hiện Q lần tưới, lần thứ j sẽ tưới Vj lít nước cho mỗi cây từ vị trí Lj đến Rj.

Sau khi hoàn thành tất cả các lần tưới, in ra lượng nước mỗi cây nhận được.

Dữ liệu vào (TUOICAY.INP):

  • Q dòng tiếp theo: Mỗi dòng chứa 3 số Lj, Rj, Vj (1 ≤ Lj ≤ Rj ≤ N; |Vj| ≤ 109)
Kết quả ra (TUOICAY.OUT):
  • Một dòng chứa N số nguyên, số thứ i là tổng lượng nước cây thứ i nhận được

// Bài 5: TUOICAY - Mảng hiệu (Difference Array)
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5 + 5;
long long diff[MAXN];

int main() {
    ifstream in("TUOICAY.INP");
    ofstream out("TUOICAY.OUT");
    
    int n, q;
    in >> n >> q;
    
    // Xử lý Q thao tác với mảng hiệu
    while(q--) {
        int l, r;
        long long v;
        in >> l >> r >> v;
        diff[l] += v;      // Bắt đầu cộng từ l
        diff[r+1] -= v;   // Ngừng cộng sau r
    }
    
    // Tính prefix sum để ra kết quả
    long long sum = 0;
    for(int i = 1; i <= n; i++) {
        sum += diff[i];
        out << sum << " ";
    }
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Mảng hiệu (A3). Hình dung: cộng V cho cả đoạn [L,R] giống như "bật vòi nước" tại L (+V) và "tắt vòi" tại R+1 (−V). Tất cả các vị trí từ L đến R đều được tưới đều V lít. Sau Q lần tưới, tính prefix sum trên mảng diff để biết mỗi cây nhận bao nhiêu nước. O(1) mỗi truy vấn, O(N) cuối cùng.

// Bài 6: DONGPHUC - Mảng hiệu + Tìm max
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5 + 5;
int diff[MAXN];

int main() {
    ifstream in("DONGPHUC.INP");
    ofstream out("DONGPHUC.OUT");
    
    int n, q;
    in >> n >> q;
    
    // Đánh dấu các đoạn bằng mảng hiệu
    while(q--) {
        int l, r;
        in >> l >> r;
        diff[l]++;
        diff[r+1]--;
    }
    
    // Tính và tìm max
    int sum = 0, ans = 0;
    for(int i = 1; i <= n; i++) {
        sum += diff[i];
        ans = max(ans, sum);
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Mảng hiệu (A3). Mỗi đợt kiểm tra đoạn [L,R] = "cộng 1" cho cả đoạn. Hình dung: mỗi đợt kiểm tra như tô màu 1 thanh ngang trên biểu đồ. diff[L]++ = bắt đầu tô, diff[R+1]−− = dừng tô. Sau Q đợt, prefix sum cho biết mỗi HS bị chồng bao nhiêu lớp. Lấy max. O(N + Q).
Bài 7: TĂNG LƯƠNG
TANGLUONG.INP / TANGLUONG.OUT

Công ty có N nhân viên đánh số từ 1 đến N, ban đầu lương đều bằng 0. Giám đốc thực hiện Q đợt tăng lương. Đợt thứ j sẽ tăng lương theo quy tắc: nhân viên L được tăng A đồng, nhân viên L+1 được tăng A+D đồng, nhân viên L+2 được tăng A+2D đồng, ..., nhân viên R được tăng A+(R-L)×D đồng.

Sau tất cả các đợt tăng lương, in ra lương của từng nhân viên.

Dữ liệu vào (TANGLUONG.INP):
  • Dòng 1: Hai số nguyên N và Q (1 ≤ N, Q ≤ 105)
  • Q dòng tiếp theo: Mỗi dòng chứa 4 số L, R, A, D (1 ≤ L ≤ R ≤ N; |A|, |D| ≤ 106)
Kết quả ra (TANGLUONG.OUT):
  • Một dòng chứa N số nguyên, số thứ i là lương của nhân viên thứ i

// Bài 7: TANGLUONG - Mảng hiệu cấp 2
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5 + 5;
long long d1[MAXN], d2[MAXN]; // Mảng hiệu cấp 1 và 2

int main() {
    ifstream in("TANGLUONG.INP");
    ofstream out("TANGLUONG.OUT");
    
    int n, q;
    in >> n >> q;
    
    while(q--) {
        int l, r;
        long long a, d;
        in >> l >> r >> a >> d;
        // Cộng cấp số cộng: a, a+d, a+2d, ...
        d1[l] += a;
        d1[r+1] -= a + (long long)(r-l)*d;
        d2[l+1] += d;
        d2[r+1] -= d;
    }
    
    long long inc = 0, sum = 0;
    for(int i = 1; i <= n; i++) {
        inc += d2[i];
        sum += d1[i] + inc;
        out << sum << " ";
    }
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Nâng cấp Mảng hiệu (A3) lên cấp 2. Bài 5–6 cộng đều (+V cho cả đoạn), ở đây cộng cấp số cộng V, V+d, V+2d,... Hình dung: lương tăng dần từ L đến R. Dùng 2 mảng: d1 đánh dấu phần hằng số (giống mảng hiệu thường), d2 đánh dấu phần "mỗi bước tăng thêm d". Khi tính: d2 tích lũy thành lượng tăng thêm, cộng vào d1, rồi prefix sum cho kết quả. O(N + Q).

// Bài 8: SONTUONG - Mảng hiệu 2D
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1005;
int diff[MAXN][MAXN];

int main() {
    ifstream in("SONTUONG.INP");
    ofstream out("SONTUONG.OUT");
    
    int m, n, q;
    in >> m >> n >> q;
    
    // Đánh dấu với mảng hiệu 2D
    while(q--) {
        int r1, c1, r2, c2;
        in >> r1 >> c1 >> r2 >> c2;
        diff[r1][c1]++;
        diff[r1][c2+1]--;
        diff[r2+1][c1]--;
        diff[r2+1][c2+1]++;
    }
    
    // Tính prefix sum 2D
    for(int i = 1; i <= m; i++) {
        for(int j = 1; j <= n; j++) {
            diff[i][j] += diff[i-1][j] + diff[i][j-1] - diff[i-1][j-1];
            if(j > 1) out << " ";
            out << diff[i][j];
        }

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("KHIEUVU.INP");
    ofstream out("KHIEUVU.OUT");
    
    int n;
    long long k;
    in >> n >> k;
    
    vector<long long> a(n);

int main() {

using namespace std;

int main() {
    ifstream in("HOPNHAT.INP");
    ofstream out("HOPNHAT.OUT");
    
    int n, m;
    in >> n >> m;
    
    vector<long long> a(n), b(m);
    for(int i = 0; i < n; i++) in >> a[i];
    for(int i = 0; i < m; i++) in >> b[i];

// Bài 13: MATKHAU - Sliding Window + Counting
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("MATKHAU.INP");
    ofstream out("MATKHAU.OUT");
    
    string s;
    int k;
    in >> s >> k;
    
    int n = s.size();
    int cnt[26] = {0};
    int distinct = 0, ans = 0;
    int left = 0;
    
    for(int right = 0; right < n; right++) {
        if(cnt[s[right] - 'a']++ == 0) distinct++;
        
        // Thu hẹp cửa sổ nếu có nhiều hơn K ký tự
        while(distinct > k) {
            if(--cnt[s[left] - 'a'] == 0) distinct--;
            left++;
        }
        
        ans = max(ans, right - left + 1);
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Sliding Window (A6) + Mảng đếm (A10). Hình dung cửa sổ co giãn: mở rộng right thêm ký tự mới, dùng mảng đếm theo dõi số loại ký tự trong cửa sổ. // Bài 14: NHIETDO - Sliding Window Max (Deque)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("NHIETDO.INP");
    ofstream out("NHIETDO.OUT");
    
    int n, k;
    in >> n >> k;
    
    vector<long long> a(n);
    for(int i = 0; i < n; i++) in >> a[i];
    
    // Deque lưu chỉ số, giảm dần theo giá trị
    deque<int> dq;
    
    for(int i = 0; i < n; i++) {
        // Loại bỏ phần tử ngoài cửa sổ
        while(!dq.empty() && dq.front() <= i - k)
            dq.pop_front();
        
        // Loại bỏ phần tử nhỏ hơn a[i]
        while(!dq.empty() && a[dq.back()] <= a[i])
            dq.pop_back();
        
        dq.push_back(i);
        
        if(i >= k - 1)
            out << a[dq.front()] << " ";
    }
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Monotonic Stack (A7) kết hợp Deque (C2). Hình dung deque như "dãy lính gác": chỉ giữ những người cao dần từ trước ra sau. #include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("ADN.INP");
    ofstream out("ADN.OUT");
    
    string s;
    in >> s;
    
    int n = s.size();
    int lastPos[128];
    memset(lastPos, -1, sizeof(lastPos));
    
    int left = 0, ans = 0;
    for(int right = 0; right < n; right++) {
        // Nếu ký tự đã xuất hiện trong cửa sổ

// Bài 17: SANXUAT - Sliding Window + 2 Deques
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("SANXUAT.INP");
    ofstream out("SANXUAT.OUT");
    
    int n, k;
    in >> n >> k;
    
    vector<long long> a(n);
    for(int i = 0; i < n; i++) in >> a[i];
    
    deque<int> maxQ, minQ; // Deque cho max và min
    long long ans = LLONG_MAX;
    
    for(int i = 0; i < n; i++) {
        // Xóa phần tử ngoài cửa sổ
        while(!maxQ.empty() && maxQ.front() <= i - k) maxQ.pop_front();
        while(!minQ.empty() && minQ.front() <= i - k) minQ.pop_front();
        
        // Duy trì deque đơn điệu
        while(!maxQ.empty() && a[maxQ.back()] <= a[i]) maxQ.pop_back();
        while(!minQ.empty() && a[minQ.back()] >= a[i]) minQ.pop_back();
        
        maxQ.push_back(i); minQ.push_back(i);
        
        if(i >= k - 1)
            ans = min(ans, a[maxQ.front()] - a[minQ.front()]);
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}

// Bài 18: THAPCAPH - Next Greater Element (Stack)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("THAPCAPH.INP");
    ofstream out("THAPCAPH.OUT");
    
    int n;
    in >> n;
    
    vector<long long> a(n), ans(n, -1);
    for(int i = 0; i < n; i++) in >> a[i];
    
    stack<int> st; // Stack lưu chỉ số
    
    for(int i = 0; i < n; i++) {
        // Xử lý các phần tử trong stack nhỏ hơn a[i]
        while(!st.empty() && a[st.top()] < a[i]) {
            ans[st.top()] = a[i];
            st.pop();
        }
        st.push(i);
    }
    
    for(int i = 0; i < n; i++)
        out << ans[i] << " ";
    
    in.close();
    out.close();
    return 0;
}

// Bài 19: BIENQC - Largest Rectangle in Histogram (Stack)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("BIENQC.INP");
    ofstream out("BIENQC.OUT");
    
    int n;
    in >> n;
    
    vector<long long> h(n);
    for(int i = 0; i < n; i++) in >> h[i];
    
    stack<int> st;
    long long ans = 0;
    
    for(int i = 0; i <= n; i++) {
        long long curr = (i == n) ? 0 : h[i];
        while(!st.empty() && h[st.top()] > curr) {
            long long height = h[st.top()];
            st.pop();
            long long width = st.empty() ? i : i - st.top() - 1;
            ans = max(ans, height * width);
        }
        st.push(i);
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}

// Bài 20: DAODONG - Contribution Technique + Stack
#include <bits/stdc++.h>
using namespace std;

int n;
long long a[100005];

// Hàm tính tổng đóng góp (isMax=1: tính max, isMax=0: tính min)
long long sumContribution(int isMax) {
    int prevSmaller[100005], nextSmaller[100005];
    stack<int> st;
    
    for(int i = 0; i < n; i++) {
        while(!st.empty() && (isMax ? a[st.top()] <= a[i] : a[st.top()] >= a[i]))
            st.pop();
        prevSmaller[i] = st.empty() ? -1 : st.top();
        st.push(i);
    }
    while(!st.empty()) st.pop();
    
    for(int i = n-1; i >= 0; i--) {
        while(!st.empty() && (isMax ? a[st.top()] < a[i] : a[st.top()] > a[i]))
            st.pop();
        nextSmaller[i] = st.empty() ? n : st.top();
        st.push(i);
    }
    
    long long sum = 0;
    for(int i = 0; i < n; i++) {
        long long left = i - prevSmaller[i];
        long long right = nextSmaller[i] - i;
        sum += a[i] * left * right;
    }
    return sum;
}

int main() {
    ifstream in("DAODONG.INP");
    ofstream out("DAODONG.OUT");
    
    in >> n;
    for(int i = 0; i < n; i++) in >> a[i];
    
    out << sumContribution(1) - sumContribution(0);
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Contribution Technique (B5) + Monotonic Stack (A7). Đảo góc nhìn: thay vì duyệt từng đoạn tìm max/min (O(N³)), hỏi ngược: a[i] là max của BAO NHIÊU đoạn? Dùng stack tìm L = số ô liên tục bên trái ≤ a[i], R = bên phải. Mọi đoạn chọn 1 ô trong L ô trái và 1 ô trong R ô phải đều có max = a[i] → số đoạn = L×R. Đóng góp của a[i] vào tổng max = a[i]×L×R. Tương tự cho min. Tổng dao động = Σ đóng góp max − Σ đóng góp min. O(N).

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("HUNGNUOC.INP");
    ofstream out("HUNGNUOC.OUT");
    
    int n;
    in >> n;
    
    vector<int> h(n);
    for(int i = 0; i < n; i++) in >> h[i];
    
    int left = 0, right = n - 1;
    int leftMax = 0, rightMax = 0;
    long long water = 0;
    
    while(left < right) {
        if(h[left] < h[right]) {
            if(h[left] >= leftMax) leftMax = h[left];
            else water += leftMax - h[left];
            left++;
        } else {
            if(h[right] >= rightMax) rightMax = h[right];
            else water += rightMax - h[right];
            right--;
        }
    }
    
    out << water;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Two Pointers ngược chiều (A5). Hình dung: nước tại vị trí i = min(tường cao nhất bên trái, bên phải) − h[i]. 2 con trỏ từ 2 đầu: bên nào THẤP hơn thì nước chắc chắn bị giới hạn bởi bên đó (vì bên kia ≥). Dịch con trỏ thấp vào, cộng nước. O(N), không cần mảng phụ.

// Bài 23: SANPHAM - Binary Search (lower/upper bound)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("SANPHAM.INP");
    ofstream out("SANPHAM.OUT");
    
    int n, q;
    in >> n >> q;
    
    vector<long long> a(n);
    for(int i = 0; i < n; i++) in >> a[i];
    
    while(q--) {
        long long x;
        in >> x;
        
        // Tìm vị trí đầu và cuối của x
        int lo = lower_bound(a.begin(), a.end(), x) - a.begin();
        int hi = upper_bound(a.begin(), a.end(), x) - a.begin();
        
        if(lo == n || a[lo] != x) {
            out << "-1 -1\n";
        } else {
            out << (lo + 1) << " " << hi << "\n";
        }
    }
    
    in.close();
    out.close();
    return 0;
}

// Bài 24: KHAITHAC - Binary Search on Answer
#include <bits/stdc++.h>
using namespace std;

int n;
long long k;
vector<long long> h;

// Tính tổng gỗ thu được khi cắt ở độ cao m
long long getWood(long long m) {
    long long sum = 0;
    for(int i = 0; i < n; i++)
        if(h[i] > m) sum += h[i] - m;
    return sum;
}

int main() {
    ifstream in("KHAITHAC.INP");
    ofstream out("KHAITHAC.OUT");
    
    in >> n >> k;
    h.resize(n);
    for(int i = 0; i < n; i++) in >> h[i];
    
    // Binary search trên kết quả
    long long lo = 0, hi = *max_element(h.begin(), h.end());
    long long ans = 0;
    
    while(lo <= hi) {
        long long mid = (lo + hi) / 2;
        if(getWood(mid) >= k) {
            ans = mid;
            lo = mid + 1;
        } else {
            hi = mid - 1;
        }
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}

// Bài 25: CHUONG - Binary Search on Answer
#include <bits/stdc++.h>
using namespace std;

vector<long long> x;
int n, c;

// Kiểm tra có thể đặt C con bò với khoảng cách tối thiểu d?
bool canPlace(long long d) {
    int cnt = 1;
    long long last = x[0];
    for(int i = 1; i < n; i++) {
        if(x[i] - last >= d) {
            cnt++;
            last = x[i];
        }
    }
    return cnt >= c;
}

int main() {
    ifstream in("CHUONG.INP");
    ofstream out("CHUONG.OUT");
    
    in >> n >> c;
    x.resize(n);
    for(int i = 0; i < n; i++) in >> x[i];
    sort(x.begin(), x.end());
    
    long long lo = 1, hi = x[n-1] - x[0], ans = 0;
    while(lo <= hi) {
        long long mid = (lo + hi) / 2;
        if(canPlace(mid)) {
            ans = mid;
            lo = mid + 1;
        } else {
            hi = mid - 1;
        }
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("GHEPCAP.INP");
    ofstream out("GHEPCAP.OUT");
    
    int n, m;
    in >> n >> m;
    
    vector<long long> a(n), b(m);
    for(int i = 0; i < n; i++) in >> a[i];
    for(int i = 0; i < m; i++) in >> b[i];
    
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());
    
    // Two pointers tìm chênh lệch nhỏ nhất
    int i = 0, j = 0;
    long long ans = LLONG_MAX;
    
    while(i < n && j < m) {
        ans = min(ans, abs(a[i] - b[j]));
        if(a[i] < b[j]) i++;

    }

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("SONGSINH.INP");
    ofstream out("SONGSINH.OUT");

        long long x;

// Bài 34: VONGTRON - Circular Kadane
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("VONGTRON.INP");
    ofstream out("VONGTRON.OUT");
    
    int n;
    in >> n;
    vector<long long> a(n);
    long long total = 0;
    
    for(int i = 0; i < n; i++) {
        in >> a[i];
        total += a[i];
    }
    
    // Kadane tìm max và min subarray
    long long maxSum = a[0], curMax = a[0];
    long long minSum = a[0], curMin = a[0];
    
    for(int i = 1; i < n; i++) {
        curMax = max(a[i], curMax + a[i]);
        maxSum = max(maxSum, curMax);
        curMin = min(a[i], curMin + a[i]);
        minSum = min(minSum, curMin);
    }
    
    // Kết quả: max(maxSum, total - minSum)
    if(total == minSum) // Tất cả âm
        out << maxSum;
    else
        out << max(maxSum, total - minSum);
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Kadane (A11) + Biến đổi bài toán (4.1). Vòng tròn nên đoạn có thể quấn qua 2 đầu (lấy đuôi + đầu). Có 2 trường hợp: (1) đoạn nằm gọn giữa mảng → Kadane thường cho maxSum; (2) đoạn quấn qua 2 đầu → phần KHÔNG chọn là 1 đoạn liên tục ở giữa = đoạn tổng nhỏ nhất → tổng chọn = Tổng − minSum. Kết quả = max(maxSum, Tổng − minSum). Lưu ý: nếu toàn âm thì Tổng − minSum = 0 (chọn rỗng), lúc đó lấy maxSum. O(N).

// Bài 35: DUAN - Hai dự án không giao nhau
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DUAN.INP");
    ofstream out("DUAN.OUT");
    
    int n;
    in >> n;
    vector<long long> a(n);
    for(int i = 0; i < n; i++) in >> a[i];
    
    // L[i] = max subarray kết thúc tại hoặc trước i
    vector<long long> L(n), R(n);
    long long curMax = a[0], maxSoFar = a[0];
    L[0] = a[0];
    for(int i = 1; i < n; i++) {
        curMax = max(a[i], curMax + a[i]);
        maxSoFar = max(maxSoFar, curMax);
        L[i] = maxSoFar;
    }
    
    // R[i] = max subarray bắt đầu tại hoặc sau i
    curMax = a[n-1]; maxSoFar = a[n-1];
    R[n-1] = a[n-1];
    for(int i = n-2; i >= 0; i--) {
        curMax = max(a[i], curMax + a[i]);
        maxSoFar = max(maxSoFar, curMax);
        R[i] = maxSoFar;
    }
    
    // Tìm max(L[i] + R[i+1])
    long long ans = LLONG_MIN;
    for(int i = 0; i < n-1; i++)
        ans = max(ans, L[i] + R[i+1]);
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Kadane (A11) + Prefix Sum (A1). Hình dung: 2 dự án không giao = chia mảng tại vị trí i: dự án trái [1..i], dự án phải [i+1..N]. Tính trước: L[i] = max subarray kết thúc ở hoặc trước i (Kadane trái→phải), R[i] = max subarray bắt đầu ở hoặc sau i (Kadane phải→trái). Kết quả = max(L[i] + R[i+1]). O(N).

// Bài 36: VUNGDAT - 2D Kadane
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("VUNGDAT.INP");
    ofstream out("VUNGDAT.OUT");
    
    int m, n;
    in >> m >> n;
    vector<vector<long long>> a(m, vector<long long>(n));
    for(int i = 0; i < m; i++)
        for(int j = 0; j < n; j++)
            in >> a[i][j];
    
    long long ans = LLONG_MIN;
    
    // Duyệt từng cặp hàng (top, bottom)
    for(int top = 0; top < m; top++) {
        vector<long long> colSum(n, 0);
        for(int bot = top; bot < m; bot++) {
            // Cộng thêm hàng bot
            for(int j = 0; j < n; j++)
                colSum[j] += a[bot][j];
            
            // Kadane 1D trên colSum
            long long curMax = colSum[0], maxSum = colSum[0];
            for(int j = 1; j < n; j++) {
                curMax = max(colSum[j], curMax + colSum[j]);
                maxSum = max(maxSum, curMax);
            }
            ans = max(ans, maxSum);
        }
    }
    
    out << ans;
    in.close();
    out.close();
    return 0;
}

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("XEPHANG.INP");
    ofstream out("XEPHANG.OUT");

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("WIFI.INP");
    ofstream out("WIFI.OUT");
    
    int n;
    in >> n;
    vector<pair<long long, long long>> intervals(n);
    
    for(int i = 0; i < n; i++)
        in >> intervals[i].first >> intervals[i].second;
    
    // Sắp xếp theo điểm bắt đầu
    sort(intervals.begin(), intervals.end());
    
    long long total = 0;
    long long curL = intervals[0].first;
    long long curR = intervals[0].second;
    
    for(int i = 1; i < n; i++) {
        if(intervals[i].first <= curR) {
            // Giao nhau, mở rộng
            curR = max(curR, intervals[i].second);
        } else {
            // Không giao, tính đoạn cũ và bắt đầu mới
            total += curR - curL;
            curL = intervals[i].first;
            curR = intervals[i].second;
        }
    }
    total += curR - curL; // Đoạn cuối
    
    out << total;
    in.close();
    out.close();
    return 0;
}

// Bài 39: TACDUONG - Sweep Line
#include <bits/stdc++.h>
using namespace std;

// Hàm so sánh: sắp theo tọa độ, nếu bằng thì +1 trước -1
bool cmp(pair<long long, int> a, pair<long long, int> b) {
    if(a.first != b.first) return a.first < b.first;
    return a.second > b.second;
}

int main() {
    ifstream in("TACDUONG.INP");
    ofstream out("TACDUONG.OUT");
    
    int n;
    in >> n;
    
    // Sự kiện: (tọđộ, +1/-1)
    vector<pair<long long, int>> events;
    for(int i = 0; i < n; i++) {
        long long l, r;
        in >> l >> r;
        events.push_back(make_pair(l, 1));  // Bắt đầu
        events.push_back(make_pair(r + 1, -1)); // Kết thúc
    }
    
    // Sắp xếp theo tọđộ
    sort(events.begin(), events.end(), cmp);
    
    int cnt = 0, maxCnt = 0;
    long long pos = 0;
    
    for(int i = 0; i < events.size(); i++) {
        cnt += events[i].second;
        if(cnt > maxCnt) {
            maxCnt = cnt;
            pos = events[i].first;
        }
    }
    
    out << pos << " " << maxCnt;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Mảng hiệu / Sweep Line (A3). Mỗi xe sống từ lúc vào đến lúc ra. Tại mỗi điểm thời gian: xe vào → +1, xe ra → −1. Dùng mảng hiệu (difference array): d[vào]++, d[ra]−−. Tính prefix sum → số xe có mặt tại mỗi thời điểm. Max của prefix sum = số xe nhiều nhất cùng lúc. O(N + T).

// Bài 40: QUANHE - Count Inversions (Merge Sort)
#include <bits/stdc++.h>
using namespace std;

long long invCount = 0;

void merge(vector<long long>& a, int l, int m, int r) {
    vector<long long> left(a.begin() + l, a.begin() + m + 1);
    vector<long long> right(a.begin() + m + 1, a.begin() + r + 1);
    
    int i = 0, j = 0, k = l;
    while(i < left.size() && j < right.size()) {
        if(left[i] <= right[j]) {
            a[k++] = left[i++];
        } else {
            a[k++] = right[j++];
            invCount += left.size() - i; // Tất cả left[i..] > right[j]
        }
    }
    while(i < left.size()) a[k++] = left[i++];
    while(j < right.size()) a[k++] = right[j++];
}

void mergeSort(vector<long long>& a, int l, int r) {
    if(l >= r) return;
    int m = (l + r) / 2;
    mergeSort(a, l, m);
    mergeSort(a, m + 1, r);
    merge(a, l, m, r);
}

int main() {
    ifstream in("QUANHE.INP");
    ofstream out("QUANHE.OUT");
    
    int n;
    in >> n;
    vector<long long> a(n);
    for(int i = 0; i < n; i++) in >> a[i];
    
    mergeSort(a, 0, n - 1);
    out << invCount;
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng Chia để trị / Merge Sort (B4). Đếm số cặp (i,j) với i < j mà a[i] > a[j] (số nghịch thế). Brute force O(N²). Dùng Merge Sort: khi merge 2 nửa đã sắp xếp, nếu a[i] > b[j] thì a[i..mid] đều > b[j] → cộng số phần tử còn lại. Mỗi lần merge đếm được nghịch thế liên nửa, không sót vì đã đếm nội bộ 2 nửa ở bước trước. O(N log N).
