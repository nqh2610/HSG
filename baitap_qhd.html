Bài 1: LEO CẦU THANG
CAUTHANG.INP / CAUTHANG.OUT

Một cầu thang có N bậc, đánh số từ 1 đến N. Mỗi bậc thứ i có điểm thưởng Ai (có thể âm). Một học sinh đứng ở mặt đất (bậc 0) và muốn leo lên bậc N. Tại mỗi bước, học sinh có thể bước lên 1 hoặc 2 bậc.

Hãy tìm tổng điểm thưởng lớn nhất mà học sinh có thể thu được khi leo từ bậc 0 lên bậc N.

Dữ liệu vào (CAUTHANG.INP):
  • Dòng 1: Số nguyên N (1 ≤ N ≤ 105)
  • Dòng 2: N số nguyên A1, A2, ..., AN (|Ai| ≤ 109)
Kết quả ra (CAUTHANG.OUT):
  • Một số nguyên duy nhất là tổng điểm thưởng lớn nhất

5
3 -1 7 2 -5

CAUTHANG.OUT

📝 Code C++ 
// Bài 1: CAUTHANG - DP cơ bản
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CAUTHANG.INP");
    ofstream out("CAUTHANG.OUT");
    
    int n;
    in >> n;
    
    vector<long long> a(n + 1), dp(n + 1);
    for(int i = 1; i <= n; i++) in >> a[i];

    // dp[i] = điểm lớn nhất khi leo đến bậc i
    dp[0] = 0;
    dp[1] = a[1];
    for(int i = 2; i <= n; i++) {
        dp[i] = max(dp[i-1], dp[i-2]) + a[i];
    }
    
    out << dp[n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ cơ bản (B3). dp[i] = điểm lớn nhất từ bậc 1 đến bậc i. Tại bậc i có 2 lựa chọn: (1) bước 1 bậc từ i−1 → dp[i−1] + a[i]; (2) nhảy 2 bậc từ i−2 (bỏ qua bậc i−1) → dp[i−2] + a[i]. Lấy max 2 cách. Hình dung: đứng ở bậc i, nhìn lại: đến từ bậc nào thuận lợi hơn thì chọn bậc đó. O(N).
Bài 2: XÂY TƯỜNG GẠCH
TUONGGACH.INP / TUONGGACH.OUT

Một bức tường hình chữ nhật kích thước 2×N cần được lát kín bằng các viên gạch 1×2 (có thể đặt ngang hoặc dọc).

Hãy đếm số cách lát khác nhau. Kết quả chia dư cho 109+7.

Dữ liệu vào (TUONGGACH.INP):
  • Dòng 1: Số nguyên N (1 ≤ N ≤ 106)
Kết quả ra (TUONGGACH.OUT):
  • Một số nguyên duy nhất là số cách lát (mod 109+7)

TUONGGACH.OUT

📝 Code C++ 
// Bài 2: TUONGGACH - Lát gạch 2xN (Fibonacci)
#include <bits/stdc++.h>
using namespace std;

const int MOD = 1e9 + 7;
long long dp[1000001];

int main() {
    ifstream in("TUONGGACH.INP");
    ofstream out("TUONGGACH.OUT");
    int n;
    in >> n;
    dp[1] = 1;
    dp[2] = 2;
    for(int i = 3; i <= n; i++)
        dp[i] = (dp[i - 1] + dp[i - 2]) % MOD;
    out << dp[n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ cơ bản (B3). f(n) = số cách lát bảng 2×n. Cột cuối có 2 cách lát: (1) 1 viên dọc → còn lại bảng 2×(n−1) = f(n−1) cách; (2) 2 viên ngang chồng lên nhau → còn lại 2×(n−2) = f(n−2) cách. Vậy f(n) = f(n−1) + f(n−2) — chính là dãy Fibonacci! Cơ sở: f(1)=1, f(2)=2. O(N).
Bài 3: ĐƯỜNG XUỐNG NÚI
XUONGNUI.INP / XUONGNUI.OUT

Một ngọn núi hình tam giác có N tầng. Tầng thứ i có i ô, ô thứ j (1 ≤ j ≤ i) chứa giá trị Ai,j. Từ ô (i,j) có thể đi xuống ô (i+1,j) hoặc (i+1,j+1).

Hãy tìm đường đi từ đỉnh (1,1) xuống tầng cuối (tầng N) sao cho tổng giá trị trên đường đi lớn nhất.

Dữ liệu vào (XUONGNUI.INP):
  • Dòng 1: Số nguyên N (1 ≤ N ≤ 1000)
  • N dòng tiếp theo: Dòng thứ i có i số nguyên Ai,1, ..., Ai,i (|Ai,j| ≤ 106)
Kết quả ra (XUONGNUI.OUT):
  • Một số nguyên duy nhất là tổng lớn nhất

// Bài 3: XUONGNUI - DP trên tam giác số
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("XUONGNUI.INP");
    ofstream out("XUONGNUI.OUT");
    int n;
    in >> n;
    vector<vector<long long>> a(n + 1, vector<long long>(n + 1, 0));
    for(int i = 1; i <= n; i++)
        for(int j = 1; j <= i; j++)
            in >> a[i][j];
    // dp[i][j] = tổng đường đi lớn nhất từ (i,j) xuống đáy
    vector<vector<long long>> dp(n + 1, vector<long long>(n + 1, 0));
    for(int j = 1; j <= n; j++) dp[n][j] = a[n][j];
    for(int i = n - 1; i >= 1; i--)
        for(int j = 1; j <= i; j++)
            dp[i][j] = a[i][j] + max(dp[i + 1][j], dp[i + 1][j + 1]);
    out << dp[1][1];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ bottom-up (B3). dp[i][j] = tổng đường đi lớn nhất từ ô (i,j) xuống đáy. Cơ sở: dp[n][j] = a[n][j] (tầng cuối). Chuyển trạng thái: dp[i][j] = a[i][j] + max(dp[i+1][j], dp[i+1][j+1]) — chọn đi xuống trái hoặc phải, lấy max. Tính từ tầng n-1 lên tầng 1. Kết quả = dp[1][1]. Vì sao bottom-up? Vì từ trên xuống có nhiều nhánh (tốn kém), từ dưới lên mỗi ô chỉ cộng 1 lần. O(N²).
Bài 4: NHẢY ĐÁ
NHAYDA.INP / NHAYDA.OUT

Có N viên đá xếp thành hàng, đánh số 1 đến N. Viên đá thứ i có độ cao Hi. Một chú ếch đứng ở viên đá 1 và muốn nhảy đến viên đá N. Mỗi bước nhảy, ếch có thể nhảy tới một trong K viên đá tiếp theo (tức từ đá i có thể nhảy đến đá i+1, i+2, ..., i+K). Chi phí nhảy từ đá i đến đá j bằng |Hi - Hj|.

Hãy tìm tổng chi phí nhỏ nhất để ếch nhảy từ đá 1 đến đá N.

Dữ liệu vào (NHAYDA.INP):
  • Dòng 1: Hai số nguyên N và K (2 ≤ N ≤ 105; 1 ≤ K ≤ 100)
  • Dòng 2: N số nguyên H1, ..., HN (1 ≤ Hi ≤ 106)
Kết quả ra (NHAYDA.OUT):
  • Một số nguyên duy nhất là tổng chi phí nhỏ nhất

// Bài 4: NHAYDA - Frog Jump K bước
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("NHAYDA.INP");
    ofstream out("NHAYDA.OUT");
    
    int n, k;
    in >> n >> k;
    
    vector<long long> h(n + 1);
    for(int i = 1; i <= n; i++) in >> h[i];
    
    // dp[i] = chi phí nhỏ nhất nhảy đến đá thứ i
    vector<long long> dp(n + 1, 1e18);
    dp[1] = 0;
    
    for(int i = 2; i <= n; i++) {
        for(int j = 1; j <= k && i - j >= 1; j++) {
            dp[i] = min(dp[i], dp[i-j] + abs(h[i] - h[i-j]));
        }
    }
    
    out << dp[n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ cơ bản (B3). dp[i] = chi phí nhỏ nhất nhảy đến đá i. Từ đá i có thể nhảy từ đá i−1, i−2, ..., i−K (tối đa K bước nhảy). Chi phí mỗi bước = |h[i]−h[từ]|. dp[i] = min(dp[i−j] + |h[i]−h[i−j]|) với j = 1..K. Giống bài 1 (leo thang) nhưng mở rộng từ 2 bước lên K bước, và có chi phí nhảy. O(N × K).
Bài 5: XẾP HÀNG MUA VÉ
MUAVE.INP / MUAVE.OUT

  • Dòng 2: N số nguyên T1, ..., TN (1 ≤ Ti ≤ 109)

// Bài 5: MUAVE - DP ghép cặp liền kề
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("MUAVE.INP");
    ofstream out("MUAVE.OUT");
    
    int n;
    in >> n;
    
    vector<long long> t(n + 1);
    for(int i = 1; i <= n; i++) in >> t[i];
    
    // dp[i] = thời gian nhỏ nhất để i người đầu mua xong vé
    vector<long long> dp(n + 1, 1e18);
    dp[0] = 0;
    dp[1] = t[1];
    
    for(int i = 2; i <= n; i++) {
        // Người i mua riêng, hoặc ghép với người i-1
        dp[i] = min(dp[i-1] + t[i], dp[i-2] + max(t[i-1], t[i]));
    }
    
    out << dp[n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ cơ bản (B3). dp[i] = thời gian nhỏ nhất cho i người đầu qua cầu. Mỗi lượt qua cầu tối đa 2 người. 2 lựa chọn: (1) người i đi 1 mình → dp[i−1] + t[i]; (2) người i đi cùng người i−1 → dp[i−2] + max(t[i−1], t[i]) (hai người đi cùng, thời gian = người chậm). Lấy min 2 cách. Giống bài leo thang nhưng chi phí khác. O(N).

// Bài 6: TROMNHA - House Robber vòng tròn
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("TROMNHA.INP");
    ofstream out("TROMNHA.OUT");
    int n;
    in >> n;
    vector<long long> a(n);
    for(int i = 0; i < n; i++) in >> a[i];

    // dp[i] = tổng lớn nhất khi xét nhà 0..i
    // dp[i] = max(dp[i-1], dp[i-2] + a[i])
    //   - dp[i-1]: KHÔNG trộm nhà i → giữ kết quả cũ
    //   - dp[i-2] + a[i]: CÓ trộm nhà i → phải bỏ nhà i-1 (lấy từ i-2)
    // → 2 nhà liền kề không bao giờ cùng bị trộm

    // Vì nhà xếp vòng tròn (nhà 0 kề nhà n-1), chia 2 TH:

    // TH1: Trộm nhà 0 → không được trộm nhà n-1 → xét [0..n-2]
    vector<long long> dp1(n);
    dp1[0] = a[0]; // chỉ có nhà 0, trộm nó
    dp1[1] = max(a[0], a[1]); // 2 nhà kề nhau, chọn nhà nào nhiều hơn
    for(int i = 2; i <= n - 2; i++)
        dp1[i] = max(dp1[i - 1], dp1[i - 2] + a[i]);

    // TH2: Không trộm nhà 0 → được trộm nhà n-1 → xét [1..n-1]
    vector<long long> dp2(n);
    dp2[1] = a[1];
    dp2[2] = max(a[1], a[2]);
    for(int i = 3; i <= n - 1; i++)
        dp2[i] = max(dp2[i - 1], dp2[i - 2] + a[i]);

    out << max(dp1[n - 2], dp2[n - 1]); // lấy max 2 TH
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ (B3) + biến đổi vòng tròn. Nhà xếp vòng tròn: nhà 1 và nhà N liền kề, không lấy được cả 2. Chia 2 trường hợp: (1) lấy nhà 1 → không lấy nhà N → giải bài House Robber trên [1..N−1]; (2) không lấy nhà 1 → giải trên [2..N]. Đáp án = max 2 TH. House Robber: dp[i] = max(dp[i−1], dp[i−2]+a[i]) — giống bài 1 (leo thang). O(N).
Bài 7: ĐOẠN CON TỔNG LỚN NHẤT
MAXSUM.INP / MAXSUM.OUT

Cho dãy N số nguyên A1, A2, ..., AN. Hãy tìm đoạn con liên tiếp (có ít nhất 1 phần tử) có tổng lớn nhất.

In ra tổng lớn nhất đó, cùng với vị trí bắt đầu và kết thúc của đoạn con (nếu nhiều đáp án, in đoạn có vị trí bắt đầu nhỏ nhất, nếu vẫn nhiều thì in đoạn ngắn nhất).

Dữ liệu vào (MAXSUM.INP):
  • Dòng 1: Số nguyên N (1 ≤ N ≤ 105)
  • Dòng 2: N số nguyên A1, ..., AN (|Ai| ≤ 109)
Kết quả ra (MAXSUM.OUT):
  • Dòng 1: Tổng lớn nhất
  • Dòng 2: Hai số L và R là vị trí bắt đầu và kết thúc

// Bài 7: MAXSUM - QHĐ đoạn con tổng lớn nhất
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("MAXSUM.INP");
    ofstream out("MAXSUM.OUT");
    int n;
    in >> n;
    vector<long long> a(n + 1), dp(n + 1);
    for(int i = 1; i <= n; i++) in >> a[i];
    // dp[i] = tổng lớn nhất đoạn con kết thúc tại i
    // dp[i] = max(a[i], dp[i-1] + a[i])
    dp[1] = a[1];
    vector<int> start(n + 1);
    start[1] = 1;
    int bestL = 1, bestR = 1;
    long long bestSum = dp[1];
    for(int i = 2; i <= n; i++) {
        if(dp[i - 1] + a[i] >= a[i]) {
            dp[i] = dp[i - 1] + a[i];
            start[i] = start[i - 1];
        } else {
            dp[i] = a[i];
            start[i] = i;
        }
        if(dp[i] > bestSum) {
            bestSum = dp[i];
            bestL = start[i];
            bestR = i;
        }
    }
    out << bestSum << "\n";
    out << bestL << " " << bestR;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ (B3). dp[i] = tổng lớn nhất đoạn con kết thúc tại i. dp[i] = max(a[i], dp[i-1] + a[i]) — "nối tiếp hay bắt đầu lại?" Nếu dp[i-1] âm thì giữ lại chỉ kéo xuống, bỏ đi bắt đầu mới tại a[i] tốt hơn. Kết quả = max(dp[i]). Lưu thêm chỉ số bắt đầu/kết thúc: khi bắt đầu lại → cập nhật start[i] = i; khi best cải thiện → cập nhật bestL, bestR. O(N).

// Bài 8: NENNHA - Paint House DP
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("NENNHA.INP");
    ofstream out("NENNHA.OUT");
    int n;
    in >> n;
    vector<vector<long long>> cost(n + 1, vector<long long>(3));
    for(int i = 1; i <= n; i++) in >> cost[i][0] >> cost[i][1] >> cost[i][2];
    // dp[i][c] = chi phí min sơn nhà 1..i, nhà i màu c
    vector<vector<long long>> dp(n + 1, vector<long long>(3, 0));
    for(int i = 1; i <= n; i++) {
        dp[i][0] = min(dp[i - 1][1], dp[i - 1][2]) + cost[i][0];
        dp[i][1] = min(dp[i - 1][0], dp[i - 1][2]) + cost[i][1];
        dp[i][2] = min(dp[i - 1][0], dp[i - 1][1]) + cost[i][2];
    }
    out << min({dp[n][0], dp[n][1], dp[n][2]});
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ cơ bản (B3). dp[i][c] = chi phí nhỏ nhất sơn nhà 1..i với nhà i màu c. 2 nhà liền kề không cùng màu, nên nhà i màu c thì nhà i−1 phải màu khác: dp[i][c] = min(dp[i−1][c']) + cost[i][c] với c'≠c. Chỉ có 3 màu nên lấy min của 2 màu còn lại. Kết quả = min(dp[n][0], dp[n][1], dp[n][2]). O(N).

    ifstream in("HOAHOP.INP");
    ofstream out("HOAHOP.OUT");

    for(int d = 1; d <= 9; d++) dp[d] = 1;
    
    for(int i = 2; i <= n; i++) {
        vector<long long> ndp(10, 0);
        for(int d = 0; d <= 9; d++) {
            for(int p = max(0, d - 2); p <= min(9, d + 2); p++) {
                ndp[d] = (ndp[d] + dp[p]) % MOD;
            }
        }
        dp = ndp;
    }
    
    long long ans = 0;
    for(int d = 0; d <= 9; d++) ans = (ans + dp[d]) % MOD;
    
    out << ans;
    in.close();
    out.close();
    return 0;
}

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CATSAT.INP");
    ofstream out("CATSAT.OUT");
    
    int L;
    in >> L;
    
    vector<long long> p(L + 1);
    for(int i = 1; i <= L; i++) in >> p[i];
    
    // dp[j] = doanh thu lớn nhất từ thanh dài j
    vector<long long> dp(L + 1, 0);
    
    for(int j = 1; j <= L; j++) {
        for(int i = 1; i <= j; i++) {
            dp[j] = max(dp[j], p[i] + dp[j - i]);

    }

    in.close();
    out.close();
    return 0;
}

// Bài 11: DOITIEN - DP đổi tiền (Unbounded Knapsack)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DOITIEN.INP");
    ofstream out("DOITIEN.OUT");
    
    int n, s;
    in >> n >> s;
    
    vector<int> c(n);
    for(int i = 0; i < n; i++) in >> c[i];
    
    // dp[j] = số xu ít nhất để đổi j đồng
    vector<int> dp(s + 1, s + 1);
    dp[0] = 0;
    
    for(int j = 1; j <= s; j++) {
        for(int i = 0; i < n; i++) {
            if(c[i] <= j && dp[j - c[i]] + 1 < dp[j])
                dp[j] = dp[j - c[i]] + 1;
        }
    }
    
    out << (dp[s] > s ? -1 : dp[s]);
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ — Unbounded Knapsack (B3). dp[j] = số xu ít nhất để đổi j đồng. Với mỗi mệnh giá c[i]: nếu dùng 1 xu c[i], số tiền còn lại = j−c[i], cần dp[j−c[i]] xu nữa, tổng = dp[j−c[i]]+1. // Bài: BALO - 0/1 Knapsack + Truy vết
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("BALO.INP");
    ofstream out("BALO.OUT");
    
    int n, W;
    in >> n >> W;
    
    vector<int> w(n + 1), v(n + 1);
    for(int i = 1; i <= n; i++) in >> w[i] >> v[i];
    
    // dp[i][j] = giá trị lớn nhất xét i vật đầu, trọng lượng <= j
    vector<vector<long long>> dp(n + 1, vector<long long>(W + 1, 0));
    
    for(int i = 1; i <= n; i++) {
        for(int j = 0; j <= W; j++) {
            dp[i][j] = dp[i-1][j]; // không lấy vật i
            if(j >= w[i] && dp[i-1][j - w[i]] + v[i] > dp[i][j])
                dp[i][j] = dp[i-1][j - w[i]] + v[i]; // lấy vật i
        }
    }
    
    out << dp[n][W] << "\n";
    
    // Truy vết: đi ngược từ dp[n][W]
    vector<int> chosen;
    int j = W;
    for(int i = n; i >= 1; i--) {
        if(dp[i][j] != dp[i-1][j]) { // vật i được chọn
            chosen.push_back(i);
            j -= w[i];
        }
    }
    reverse(chosen.begin(), chosen.end());
    for(int x : chosen) out << x << " ";
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ — Ba lô 0/1 (B3) + truy vết. Cần bảng 2D dp[i][j] (không tối ưu 1D) để truy vết. dp[i][j] = giá trị max xét i vật đầu, trọng lượng ≤ j. Truy vết: từ dp[n][W] đi ngược: nếu dp[i][j] ≠ dp[i−1][j] → vật i được chọn, lùi j -= w[i]; ngược lại bỏ qua vật i. O(N × W). Kỹ thuật truy vết là kỹ năng quan trọng vì HSG thường yêu cầu in phương án tối ưu, không chỉ giá trị.

// Bài 13: CHIAKEO - Subset Sum (DP)
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CHIAKEO.INP");
    ofstream out("CHIAKEO.OUT");
    
    int n;
    in >> n;
    
    vector<int> a(n);
    int sum = 0;
    for(int i = 0; i < n; i++) {
        in >> a[i];
        sum += a[i];
    }
    
    if(sum % 2 != 0) {
        out << "NO";
    } else {
        int half = sum / 2;
        // dp[j] = có thể chọn tổng đúng j không?
        vector<bool> dp(half + 1, false);
        dp[0] = true;
        
        for(int i = 0; i < n; i++) {
            for(int j = half; j >= a[i]; j--) {
                if(dp[j - a[i]]) dp[j] = true;
            }
        }
        
        out << (dp[half] ? "YES" : "NO");
    }
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ — Subset Sum (B3). Chia đều 2 nhóm = mỗi nhóm tổng = sum/2. Nếu sum lẻ → không thể. Quy về: có tập con nào tổng đúng sum/2 không? dp[j] = true/false (có thể tạo tổng j). // Bài: XAUCHUNG - LCS + Truy vết
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("XAUCHUNG.INP");
    ofstream out("XAUCHUNG.OUT");
    
    string s, t;
    in >> s >> t;
    
    int m = s.size(), n = t.size();
    // dp[i][j] = độ dài LCS của s[1..i] và t[1..j]
    vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));
    
    for(int i = 1; i <= m; i++) {
        for(int j = 1; j <= n; j++) {
            if(s[i-1] == t[j-1])
                dp[i][j] = dp[i-1][j-1] + 1;
            else
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
        }
    }
    
    out << dp[m][n] << "\n";
    
    // Truy vết: đi từ dp[m][n] ngược về dp[0][0]
    string lcs = "";
    int i = m, j = n;
    while(i > 0 && j > 0) {
        if(s[i-1] == t[j-1]) {
            lcs = s[i-1] + lcs; // ký tự chung, thêm vào đầu
            i--; j--;
        } else if(dp[i-1][j] > dp[i][j-1])
            i--; // đi lên
        else
            j--; // đi trái
    }
    out << lcs;
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ — LCS (B3) + truy vết. Tính dp[i][j] như cũ. Truy vết: từ (m,n) đi ngược: nếu s[i]==t[j] → ký tự này nằm trong LCS, thêm vào kết quả, đi chéo (i−1,j−1); nếu dp[i−1][j] > dp[i][j−1] → đi lên; ngược lại → đi trái. Hình dung: bò ngược trên bảng DP từ góc dưới phải về góc trên trái. O(M × N).

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("SUACHUOI.INP");
    ofstream out("SUACHUOI.OUT");
    
    string s, t;
    in >> s >> t;
    
    int m = s.size(), n = t.size();

    vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));
    
    for(int i = 0; i <= m; i++) dp[i][0] = i;
    for(int j = 0; j <= n; j++) dp[0][j] = j;

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("LUOI.INP");
    ofstream out("LUOI.OUT");
    
    int m, n;
    in >> m >> n;
    
    vector<vector<long long>> a(m + 1, vector<long long>(n + 1));
    // dp[i][j] = tổng lớn nhất đi từ (1,1) đến (i,j)
    vector<vector<long long>> dp(m + 1, vector<long long>(n + 1, -1e18));
    
    for(int i = 1; i <= m; i++)
        for(int j = 1; j <= n; j++)
            in >> a[i][j];

    for(int i = 1; i <= m; i++) {
        for(int j = 1; j <= n; j++) {

            if(i > 1) best = max(best, dp[i-1][j]);
            if(j > 1) best = max(best, dp[i][j-1]);
            dp[i][j] = best + a[i][j];
        }
    }
    
    out << dp[m][n];
    in.close();
    out.close();
    return 0;
}

                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
        }
    }
    
    out << dp[n][n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ — LCS (B3). Nhận xét: xâu con đối xứng dài nhất (LPS) của S chính là LCS của S và reverse(S). Vì sao? Xâu đối xứng đọc xuôi = đọc ngược, nên nó xuất hiện trong cả S lẫn reverse(S) như xâu con chung. LCS của 2 xâu dài N → bảng N×N. O(N²).

// Bài 20: CHIANHOM - DP chia đoạn K nhóm
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("CHIANHOM.INP");
    ofstream out("CHIANHOM.OUT");
    
    int n, k;
    in >> n >> k;
    
    vector<int> a(n + 1);
    for(int i = 1; i <= n; i++) in >> a[i];
    
    // Tính trước cost[i][j] = max - min trong đoạn i..j
    vector<vector<long long>> cost(n + 1, vector<long long>(n + 1, 0));
    for(int i = 1; i <= n; i++) {
        int mn = a[i], mx = a[i];
        for(int j = i; j <= n; j++) {
            mn = min(mn, a[j]);
            mx = max(mx, a[j]);
            cost[i][j] = mx - mn;
        }
    }
    
    // dp[g][i] = chi phí min chia i người đầu thành g nhóm
    vector<vector<long long>> dp(k + 1, vector<long long>(n + 1, 1e18));
    dp[0][0] = 0;
    
    for(int g = 1; g <= k; g++) {
        for(int i = g; i <= n; i++) {
            for(int j = g - 1; j < i; j++) {
                dp[g][i] = min(dp[g][i], dp[g-1][j] + cost[j+1][i]);
            }
        }
    }
    
    out << dp[k][n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ chia đoạn (B3). Chuyển tiếp: thử mọi điểm cắt j: g−1 nhóm cho 1..j, nhóm thứ g là j+1..i. dp[g][i] = min(dp[g−1][j] + cost[j+1][i]). Giống bài 25 (Đặt trạm). O(K × N²).

// Bài: TIECTY - Tree DP (tập độc lập trên cây)
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
vector<int> children[MAXN];
long long h[MAXN];
long long dp[MAXN][2]; // dp[u][0] = không mời u, dp[u][1] = mời u

void dfs(int u) {
    dp[u][0] = 0;     // không mời u
    dp[u][1] = h[u];   // mời u, được h[u] điểm
    
    for(int i = 0; i < (int)children[u].size(); i++) {
        int v = children[u][i];
        dfs(v);
        dp[u][0] += max(dp[v][0], dp[v][1]); // không mời u → con v tuỳ ý
        dp[u][1] += dp[v][0];                // mời u → con v KHÔNG được mời
    }
}

int main() {
    ifstream in("TIECTY.INP");
    ofstream out("TIECTY.OUT");
    
    int n;
    in >> n;
    
    for(int i = 1; i <= n; i++) in >> h[i];
    
    vector<bool> hasParent(n + 1, false);
    for(int i = 0; i < n - 1; i++) {
        int u, v;
        in >> u >> v;
        children[u].push_back(v);
        hasParent[v] = true;
    }
    
    // Tìm gốc (đỉnh không có cha)
    int root = 1;
    for(int i = 1; i <= n; i++)
        if(!hasParent[i]) { root = i; break; }
    
    dfs(root);
    out << max(dp[root][0], dp[root][1]);
    
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ trên cây (B3). Mỗi nút u có 2 trạng thái: mời (1) hoặc không mời (0). dp[u][1] = h[u] + Σ dp[con][0] — mời u thì tất cả con trực tiếp KHÔNG được mời. dp[u][0] = Σ max(dp[con][0], dp[con][1]) — không mời u thì mỗi con tuỳ ý. DFS từ gốc xuống lá, tính dp từ lá lên gốc (post-order). Đáp án = max(dp[root][0], dp[root][1]). O(N).

#include <bits/stdc++.h>

int main() {
    ifstream in("MATRIX.INP");
    ofstream out("MATRIX.OUT");
    
    int n;
    in >> n;

    for(int i = 0; i <= n; i++) in >> p[i];

    vector<vector<long long>> dp(n + 1, vector<long long>(n + 1, 0));

        for(int i = 1; i <= n - len + 1; i++) {

// Bài 24: CONGVIEC - Weighted Job Scheduling
#include <bits/stdc++.h>
using namespace std;

struct Job {
    int s, e;
    long long v;
};

bool cmpEnd(const Job &a, const Job &b) {
    return a.e < b.e;
}

int main() {
    ifstream in("CONGVIEC.INP");
    ofstream out("CONGVIEC.OUT");
    
    int n;
    in >> n;
    
    vector<Job> jobs(n);
    for(int i = 0; i < n; i++)
        in >> jobs[i].s >> jobs[i].e >> jobs[i].v;
    
    sort(jobs.begin(), jobs.end(), cmpEnd);
    
    // dp[i] = lợi nhuận max xét đến công việc i
    vector<long long> dp(n, 0);
    dp[0] = jobs[0].v;
    
    for(int i = 1; i < n; i++) {
        dp[i] = max(dp[i-1], jobs[i].v);
        // Tìm công việc cuối không chồng chéo bằng chặt nhị phân
        int lo = 0, hi = i - 1, best = -1;
        while(lo <= hi) {
            int mid = (lo + hi) / 2;
            if(jobs[mid].e <= jobs[i].s) {
                best = mid;
                lo = mid + 1;
            } else {
                hi = mid - 1;
            }
        }
        if(best >= 0)
            dp[i] = max(dp[i], dp[best] + jobs[i].v);
    }
    
    out << dp[n-1];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ (B3) + Chặt nhị phân (A8). Sắp xếp theo thời gian kết thúc. dp[i] = lợi nhuận lớn nhất xét i công việc đầu. 2 lựa chọn: (1) bỏ công việc i → dp[i−1]; (2) làm công việc i → cần tìm công việc gần nhất kết thúc trước khi i bắt đầu (gọi là last) bằng chặt nhị phân, được dp[last] + v[i]. Lấy max 2 cách. Vì sao cần sắp xếp? Để chặt nhị phân tìm last nhanh O(log N). O(N log N).

#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("TRAM.INP");
    ofstream out("TRAM.OUT");
    
    int n, k;
    in >> n >> k;
    
    vector<long long> x(n + 1);
    for(int i = 1; i <= n; i++) in >> x[i];
    
    // cost[i][j] = chi phí đặt 1 trạm phục vụ nhà i..j (đặt ở median)
    vector<vector<long long>> cost(n + 1, vector<long long>(n + 1, 0));

        for(int j = i; j <= n; j++) {
            int mid = (i + j) / 2;
            for(int t = i; t <= j; t++)
                cost[i][j] += abs(x[t] - x[mid]);
        }
    }
    
    // dp[g][i] = chi phí min dùng g trạm phục vụ nhà 1..i
    vector<vector<long long>> dp(k + 1, vector<long long>(n + 1, 1e18));
    dp[0][0] = 0;
    
    for(int g = 1; g <= k; g++) {
        for(int i = 1; i <= n; i++) {
            for(int j = 0; j < i; j++) {
                dp[g][i] = min(dp[g][i], dp[g-1][j] + cost[j+1][i]);
            }
        }
    }
    
    out << dp[k][n];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ chia đoạn (B3). Tính trước cost[i][j] = chi phí nếu 1 trạm phục vụ nhà i..j. Đặt trạm ở đâu tốt nhất? Ở trung vị (vị trí giữa) vì tổng khoảng cách nhỏ nhất. dp[g][i] = chi phí tối ưu dùng g trạm cho nhà 1..i. Chuyển tiếp: thử mọi điểm cắt j: g−1 trạm cho nhà 1..j, trạm thứ g cho nhà j+1..i. dp[g][i] = min(dp[g−1][j] + cost[j+1][i]). O(K × N²).

// Bài 26: ROBOTVANG - DP 2 robot trên lưới
#include <bits/stdc++.h>
using namespace std;

int a[55][55];
int dp[55][55][55]; // dp[i][j1][j2]: hàng i, robot A cột j1, robot B cột j2

int main() {
    ifstream in("ROBOTVANG.INP");
    ofstream out("ROBOTVANG.OUT");
    
    int m, n;
    in >> m >> n;
    for(int i = 1; i <= m; i++)
        for(int j = 1; j <= n; j++)
            in >> a[i][j];
    
    memset(dp, -1, sizeof(dp));
    dp[1][1][n] = a[1][1] + a[1][n];
    
    int dx[] = {-1, 0, 1};
    
    for(int i = 2; i <= m; i++) {
        for(int j1 = 1; j1 <= n; j1++) {
            for(int j2 = j1; j2 <= n; j2++) {
                for(int d1 = 0; d1 < 3; d1++) {
                    for(int d2 = 0; d2 < 3; d2++) {
                        int pj1 = j1 + dx[d1];
                        int pj2 = j2 + dx[d2];
                        if(pj1 < 1 || pj1 > n || pj2 < 1 || pj2 > n) continue;
                        int a1 = min(pj1, pj2), a2 = max(pj1, pj2);
                        if(dp[i-1][a1][a2] < 0) continue;
                        int val = a[i][j1] + (j1 != j2 ? a[i][j2] : 0);
                        dp[i][j1][j2] = max(dp[i][j1][j2], dp[i-1][a1][a2] + val);
                    }
                }
            }
        }
    }
    
    int ans = 0;
    for(int j1 = 1; j1 <= n; j1++)
        for(int j2 = j1; j2 <= n; j2++)
            if(dp[m][j1][j2] > ans) ans = dp[m][j1][j2];
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ 2 robot (B3). 2 robot đi từ hàng 1 xuống hàng M. dp[i][j1][j2] = tổng vàng lớn nhất khi ở hàng i, robot A cột j1, B cột j2. Mỗi robot có 3 hướng (trái, thẳng, phải) → 2 robot = 3×3 = 9 tổ hợp. Vì sao j1 ≤ j2? Vì 2 robot đối xứng, A bên trái B bên phải không mất tổng quát (giảm nửa số trạng thái). Nếu cùng ô chỉ lấy 1 lần. // Bài: PHANCONG - Bitmask DP phân công việc
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("PHANCONG.INP");
    ofstream out("PHANCONG.OUT");
    
    int n;
    in >> n;
    
    vector<vector<int>> c(n, vector<int>(n));
    for(int i = 0; i < n; i++)
        for(int j = 0; j < n; j++)
            in >> c[i][j];
    
    // dp[mask] = chi phí min khi đã giao các công việc trong mask
    // Người thứ k = số bit 1 trong mask (đếm từ người 0)
    vector<long long> dp(1 << n, 1e18);
    dp[0] = 0;
    
    for(int mask = 0; mask < (1 << n); mask++) {
        if(dp[mask] >= 1e18) continue;
        
        // Người tiếp theo cần giao việc = số bit 1 trong mask
        int k = __builtin_popcount(mask);
        if(k >= n) continue;
        
        // Thử giao công việc j cho người k
        for(int j = 0; j < n; j++) {
            if(mask & (1 << j)) continue; // việc j đã giao
            int nmask = mask | (1 << j);
            dp[nmask] = min(dp[nmask], dp[mask] + c[k][j]);
        }
    }
    
    out << dp[(1 << n) - 1];
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ Bitmask (B3). Dùng mask (số nhị phân N bit) để biểu diễn tập công việc đã giao. Bit thứ j = 1 nghĩa là công việc j đã được giao. dp[mask] = chi phí min khi tập công việc đã giao = mask. Người đang xét = số bit 1 trong mask (vì giao theo thứ tự người 0, 1, 2...). Thử giao mỗi công việc j chưa giao cho người hiện tại. Đáp án = dp[(1<N). Bitmask DP là kỹ thuật quan trọng khi N nhỏ (≤ 20).

// Bài: DULICH - TSP bằng Bitmask DP
#include <bits/stdc++.h>
using namespace std;

int main() {
    ifstream in("DULICH.INP");
    ofstream out("DULICH.OUT");
    
    int n;
    in >> n;
    
    vector<vector<int>> d(n, vector<int>(n));
    for(int i = 0; i < n; i++)
        for(int j = 0; j < n; j++)
            in >> d[i][j];
    
    // dp[mask][i] = chi phí min đi qua các thành phố trong mask, hiện ở thành phố i
    vector<vector<long long>> dp(1 << n, vector<long long>(n, 1e18));
    dp[1][0] = 0; // xuất phát từ thành phố 0 (đánh số từ 0)
    
    for(int mask = 1; mask < (1 << n); mask++) {
        for(int u = 0; u < n; u++) {
            if(!(mask & (1 << u))) continue; // u không trong mask
            if(dp[mask][u] >= 1e18) continue;
            
            // Thử đi đến thành phố v chưa thăm
            for(int v = 0; v < n; v++) {
                if(mask & (1 << v)) continue; // v đã thăm
                int nmask = mask | (1 << v);
                dp[nmask][v] = min(dp[nmask][v], dp[mask][u] + d[u][v]);
            }
        }
    }
    
    // Quay về thành phố 0
    int full = (1 << n) - 1;
    long long ans = 1e18;
    for(int u = 1; u < n; u++)
        ans = min(ans, dp[full][u] + d[u][0]);
    
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ Bitmask (B3). Bài toán người du lịch (TSP) — tìm chu trình Hamilton ngắn nhất. dp[mask][i] = chi phí min đi qua tập thành phố mask, hiện đang ở i. Xuất phát: dp[{0}][0] = 0. Chuyển tiếp: từ (mask, u) đi đến v chưa thăm → dp[mask|{v}][v] = dp[mask][u] + d[u][v]. Cuối: min(dp[full][u] + d[u][0]) — quay về. Giống bài phân công nhưng thêm chiều "đang ở đâu". O(N² × 2N).

// Bài 29: DUONGCAY - Đường kính cây (Tree DP)
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 100005;
vector<pair<int,int>> adj[MAXN];
long long dp[MAXN]; // dp[u] = đường đi dài nhất từ u xuống cây con
long long ans;

void dfs(int u, int parent) {
    dp[u] = 0;
    for(int i = 0; i < (int)adj[u].size(); i++) {
        int v = adj[u][i].first;
        int w = adj[u][i].second;
        if(v == parent) continue;
        dfs(v, u);
        // Đường kính qua u = nhánh dài nhất + nhánh dài nhì
        ans = max(ans, dp[u] + dp[v] + w);
        dp[u] = max(dp[u], dp[v] + w);
    }
}

int main() {
    ifstream in("DUONGCAY.INP");
    ofstream out("DUONGCAY.OUT");
    int n;
    in >> n;
    for(int i = 0; i < n - 1; i++) {
        int u, v, w;
        in >> u >> v >> w;
        adj[u].push_back(make_pair(v, w));
        adj[v].push_back(make_pair(u, w));
    }
    ans = 0;
    dfs(1, 0);
    out << ans;
    in.close();
    out.close();
    return 0;
}
💡 Ý tưởng: Dùng QHĐ trên cây (B3). dp[u] = đường đi dài nhất từ u xuống cây con của u. Với mỗi nút u, đường kính đi qua u = 2 nhánh dài nhất: dp[con1] + w1 + dp[con2] + w2. Khi DFS, mỗi lần xét con v mới, cập nhật đáp án = dp[u] (nhánh dài nhất trước đó) + dp[v] + w (nhánh hiện tại), rồi cập nhật dp[u] = max(dp[u], dp[v]+w). Đáy đệ quy: lá → dp = 0. O(N).
